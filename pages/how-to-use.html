ImgR
How to Use

If you have been building responsive websites for a while you must have encountered problems with Images

The meta viewport tag and Css styling gives us options to control the image tag, not the image file. However the image tag srcset attribute allows us specify what image to download depending on the device.

With this we can specify different resolutions or sizes of the same image depending on the device width. But how do we get different sizes of the same image?

I built this tool for just that, I’ve been using it for a while but decided to open source it. At it’s core, it just spits out mobile, tablet and desktop sizes of images, very suitable for cover images and background images.

However if your image has solid objects in them like human faces, etc. they can get distorted so I added the option to scale down your image by a factor. There’s also an option to specify individual width and height or one of either and allow the image maintain aspect ratio.
